Lecteur de fichiers WAV

Caractéristiques des fichiers WAV à utiliser :

- Fréquence échantillonnage	: 22050 Hz
- Format			: 8 bits non signés
- Nombre de canaux		: Mono

Utiliser le script Stream_WAV.py pour streamer des fichiers
